# Code of Conduct

## Our pledge

We want this project to be useful, direct, and welcoming. Contributors should be able to disagree about product and code without turning the repo into a tire fire.

## Expected behavior

- Be respectful and specific.
- Critique code and decisions, not people.
- Prefer reproducible examples over vibes.
- Assume good intent, but fix real harm quickly.
- Keep issues and PRs on topic.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Sexist, racist, homophobic, transphobic, ableist, or otherwise exclusionary comments.
- Publishing private information without consent.
- Spam, trolling, or repeated off-topic disruption.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block contributors; or take other action needed to keep the project healthy.

Report conduct problems privately to the maintainers.
